Python.SK spravuje SPy o.z.

Poslaním občianskeho združenia SPy (skratka "Slovak Python") je podpora vývojárov a používateľov programovacieho jazyka Python. Podporujeme a šírime používanie programovacieho jazyka Python a iných open source technológii a open source myšlienok v našej spoločnosti.

Viac o nás nájdete na stránke našeho občianskeho združenia: https://spy.python.sk

Python na Slovensku

Python je moderný programovací jazyk. Je univerzálny poháňa weby, ale aj robotov. Je výborný pre vyučovanie programovania.

Komunita na slovensku je mladá a aktívna. Najdeš tu pomoc, kamarátov, alebo aj prácu. Vitaj medzi nami.

Zapoj sa!

Pomôž nám rozšíriť Python na Slovensku. Pro leňochov s peniazmi máme číslo účtu, ale viac oceníme, keď sa chopíš niektorej úlohy a dokončíš ju do konca. Večná sláva zaručená!
